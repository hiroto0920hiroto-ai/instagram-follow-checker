# スマホだけで使う方法

このツールは静的サイトなので、GitHub Pages / Netlify / Vercel などにこのフォルダを置けばスマホだけで使える。

## 必要なもの

- HTTPSで開ける公開URL
- Instagram OAuthのリダイレクトURLに、その公開URLを登録
- 必要ならトークン交換用のサーバーAPI

## 使い方

1. このフォルダを静的ホスティングにアップロード
2. スマホで公開URLを開く
3. Safari / Chrome からホーム画面に追加
4. Instagramログイン設定を入れて使う

PCをサーバーとして起動する必要はない。
